##Python 斜拋運動模擬與分析器（含風向風速）

##一、專案目的
本專案的目的是活用課堂所學之斜拋運動物理概念，搭配Python程式設計，
建立一個可以模擬物體斜拋運動的程式。
此外，為了讓模擬更接近真實狀況，本專案加入「風向與風速」的影響，
觀察其對運動軌跡的改變。

##二、程式功能與技術原理
(1) 功能
  使用者輸入初速度、發射角度與重力加速度
  加入水平與垂直方向的風速
  計算：
    飛行時間
    最大高度
    最大水平射程
使用 matplotlib 繪製運動軌跡圖
使用 matplotlib.animation 繪製運動軌跡圖

(2) 技術原理
    將初速度分解為水平與垂直分量
    使用時間離散法（Euler method）逐步更新位置
    假設風速為常數，直接加在速度分量上，不考慮空氣阻力平方項
x(t)=v0​*cosθ*t
y(t)=v0*​sinθ*t−2/1​gt**2
飛行時間：t​=2*v0*​sinθ/g
最大高度：H​=(v0​sinθ)**2/2g
最大水平射程:R​=v0**2​*sin2θ/g
加入風力影響：
假設風在水平 x 方向，修正水平速度：
vx​=v0*​cosθ+vxwind
vy​=v0​sinθ−gt+vywind​s
更新位置：
x(t+Δt)=x(t)+vx​Δt
y(t+Δt)=y(t)+vy​Δt

##三、使用方式
1.執行程式
2.依序輸入：
    初速度（m/s）
    發射角度（degree）
    重力加速度（m/s²）
    水平風速（m/s）
    垂直風速（m/s）
3.程式會輸出計算結果並顯示運動軌跡圖,比較不同軌跡的運動軌跡

##四、程式架構
使用 numpy 進行數值計算
使用 matplotlib  進行繪圖
主程式包含：
    參數輸入
    速度分解
    迴圈模擬運動
    結果計算與視覺化
    多軌跡比較
    動畫模擬

##五、開發過程與問題紀錄
在開發過程中遇到幾個問題：

    1.一開始不知道如何判斷物體落地  
    解決方式：設定 y < 0 時停止迴圈

    2.一開始我的程式只能模擬一組參數，我不確定要怎麼在同一張圖上畫多條線，因此向 AI 詢問解法。AI 建議我將每條軌跡的 x 與 y 資料存成 list，再用 for 迴圈逐條繪製，讓我在不大幅修改原本程式的情況下完成多軌跡比較。


    3.時間間隔設太大導致軌跡不平滑  
    將dt調小為 0.01 秒

    4.在程式開發過程中，我向 AI 詢問如何加入風速影響。
    AI建議使用時間離散法逐步更新位置，使程式邏輯更清楚。

    5.動畫部分一開始只會畫靜態圖，AI向我說明 matplotlib animation 的基本概念，我再依照自己的程式結構進行調整。
